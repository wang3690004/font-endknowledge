###云 公有云 私有云 混合云基本概念
前端大概率也接触不到云相关的开发和运用，只是学习 整理记录下学习云基本概念懂一点行业其他知识,仅此而已

为什么需要云？
```
传统的应用越来越复杂，需要更为强大的服务器，企业要自己买各种服务器等硬件设备，并招募更强大的运维团队。
这里面的成本是巨大的 小公司hold不住，所以一种新的模式应运而生 云计算，将应用部署到云端后，可以不必操心硬件问题，云厂商帮你自动负载均衡，集群分布 都帮你弄好了，而且使用起来非常稳定安全。
小公司只需要给服务费就可以了 这对小公司而言是一个不错的轻资产方式,更像是个共享经济 让硬件设备的利用率达到更大
```

什么是云？
```
狭义上：提供一种资源网络，使用者可以随时获取云上的资源，按需求量使用，就像自来水厂一样 随时接水 不限量，付水费就行了
广义上: 把计算资源集合起来，通过软件实现自动化管理，只需要很少的人参与，就能让资源被快速提供。把计算能力作为一种商品，可以再互联网上像水电一样 方便使用
```

云计算特点?
1. 大规模，分布式
```
云厂商资金和实力雄厚，可以完全满足需求
```
2. 虚拟化
```
云计算采用虚拟化技术,用户不需要关注具体硬件实体，配置下服务直接用就可以了。
```
3. 高可用性和扩展性
```
云厂商一半会采用数据多副本容错 计算节点同构可互换等措施来保障服务的高可靠性，另外还可以动态伸缩规模，自动扩容，负载均衡 分布式计算 效用计算 并行计算 网络存储 备份等方式让资源可以得到最大的利用
```
4. 按需服务 更加经济
```
用户根据自己的需求购买服务
```
5. 安全
```
网络安全已经成为企业或个人创业者面对的问题，企业的it团队或个人很难应付那些来自网络的攻击 使用云服务可以借助更专业的团队降低风险
```
类型?
1. 公有云:最基础的服务，成本较低 多个客户共享一个服务器提供商的系统资源，他们无需架构任何设备配备任何管理人员，便可以享有专业的it服务 
2. 私有云：一些行业对客户隐私极为看重的。他们不可能降重要数据放到公共网络上，所以喜欢自己架设私有云端网络，总体上看 私有云的运作形式和公有云类似。但是企业需要自行设计数据中心，网络 存储设备 并拥有专业顾问团队。
3. 混合云，既有公有云 又有私有云，重要的敏感的用私有 不重要的用公有

服务类别?
laas:基础设施即服务，把计算基础（服务器 网络技术 存储数据中心等）作为一项服务卖给客户，当然也包括提供系统和虚拟化技术，来管理资源。说白了就是给你提供一个空硬件 里面各种东西需要你自己去安装。
paas:平台即服务，实际上就是指降软件研发的平台作为一种服务，供应商提供超过基础设施的服务，一个作为软件开发和运行环境的整套解决方案，这个是再硬件基础上做一个新的扩展，基础软件和环境都给你提供了 给你一个平台 剩下的全靠自己去发挥。
saas:软件即服务,是一种交付模式，其中应用作为一项服务托管 通过网络提供给客户，帮助客户更好的管理他们的it项目和服务 确保他们it应用的质量和性能，监控他们的在线业务。从硬件到软件不用费力
