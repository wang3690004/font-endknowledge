####杂记
1.开发过程中包括vue，react都会存在这种简易写法，之前一直误会，这次记录下来。
```
let a ={
    s(){
    }
}
let b = {
    s:function(){

    }
}
//a {s:f}     b{s:f}  其实两者是一样的效果的
```
正如现在的vue项目遇到的computed和watch写法是类似的
```
watch:{
    value(val){
        this.a = val
    },
    visiblesync(newval){
        this.$emit('input',newval);
    }
},
computed:{
    language(){
        return this.$i18m.locale
    }
}
```
